# Liquid AI (liquid-ai)

Liquid AI is an MIT spinoff developing Liquid Foundation Models (LFMs) — a new class of generative models based on liquid neural networks. Offers LFM-1B, LFM-3B, LFM-40B with on-prem and edge deployment options.
